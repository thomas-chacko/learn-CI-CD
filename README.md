# CI/CD with React

This repository is designed to learn CI/CD (Continuous Integration/Continuous Deployment) using GitHub Actions.

## Overview

Learn how to automate testing, building, and deploying React applications through GitHub Actions workflows.

## Getting Started

- Clone this repository
- Install dependencies with `npm install`
- Explore the `.github/workflows` directory to see CI/CD configurations
